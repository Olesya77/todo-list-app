# Документация к хранилищу данных

Приложение «Менеджер списка дел» сохраняет данные в файл `tasks.json`.

## Формат JSON
{
  "tasks": [
    {
      "id": 1,
      "title": "Купить молоко",
      "done": false
    },
    {
      "id": 2,
      "title": "Сдать проект",
      "done": true
    }
  ]
}

## Описание полей
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | число | Уникальный номер задачи |
| `title` | строка | Текст задачи |
| `done` | логический | `true` – выполнено, `false` – не выполнено |

## Принцип работы
- При запуске приложения файл читается, задачи загружаются.
- При любом изменении (добавление, удаление, редактирование, отметка о выполнении) файл перезаписывается.
- Если файла нет или он повреждён, создаётся пустой список.
